# nuget-trivia

`nuget-trivia` provides reusable question-model packages and Open Trivia DB integration for .NET applications.

Use this package family if you want to build quiz, exam, or demo scenarios on shared trivia contracts instead of application-local models.

## Packages

- `Italbytz.Trivia.Abstractions`
- `Italbytz.Trivia.OpenTriviaDb`

## What you can do with nuget-trivia

- define reusable question, answer, and difficulty models
- call Open Trivia DB through a dedicated package client
- reuse trivia-oriented building blocks across teaching, quiz, and exam scenarios
